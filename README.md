## **TodoList**

TodoList — приложение для ведения списка задач. Пользователь создаёт задачи, редактирует их, отмечает выполненными и быстро находит нужные через поиск. При первом запуске приложение подгружает тестовые задачи из сети и сохраняет их локально.


## **Описание приложения**

- Приложение-список задач, в котором пользователь создаёт задачи и управляет ими.
- Каждая задача содержит название, описание и дату создания.
- На главном экране отображается список задач с поиском по названию и описанию.
- Задачи можно отмечать выполненными, редактировать, удалять и делиться ими.
- Если локальных задач нет, приложение загружает их из сети и сохраняет в Core Data.

## **Функциональные требования**

- Создание новой задачи;
- Редактирование существующей задачи;
- Удаление задачи через контекстное меню;
- Отметка выполнения задачи (повторное нажатие снимает отметку);
- Поиск по названию и описанию задачи;
- Поделиться задачей через системное меню;
- Отображение количества задач и состояния пустого списка;
- Загрузка задач из сети при пустом локальном хранилище;
- Хранение задач локально между запусками.

## **Технические требования**

- Язык реализации — Swift; UI-фреймворк — UIKit;
- Архитектура — VIPER (View/Interactor/Presenter/Router);
- Хранение данных — Core Data;
- Сетевой слой — URLSession, async/await.
